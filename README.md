##  Logger Service
這是一個模擬python client打到server，並可以在資料庫新增data的一套程式

### client運行指引
Python  
``python client1.py``  

C 
``sudo dnf install libcurl-devel #安裝所需函式庫``
```
gcc -o client2 client2.c -lcurl  
./client2
```

Java  
``dnf install java-1.8.0-openjdk-devel``
```
javac client3.java  
java client3  
```
